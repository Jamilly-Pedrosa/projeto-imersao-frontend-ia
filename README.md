# — .✦ Projeto Imersão Front-End com IA 🎬

Este projeto foi desenvolvido durante uma imersão prática de Front-End com o apoio de Inteligência Artificial, com foco na construção de interfaces modernas, organizadas e responsivas.

O objetivo foi aplicar conceitos fundamentais de desenvolvimento web utilizando HTML, CSS e JavaScript, além de explorar como ferramentas de IA, como o GitHub Copilot, podem auxiliar no processo de desenvolvimento.

---

## .✦ Objetivo do projeto

Reproduzir uma interface inspirada na Netflix, incluindo tela de seleção de perfis e catálogo, utilizando Inteligência Artificial como apoio durante o desenvolvimento.

---

## .✦ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Gemini (IA) – geração de imagens
- GitHub Copilot (VS Code)

## .✦ O que foi aprendido

### Aula 1 - Estrutura inicial

- Configuração do ambiente com Live Server
- Criação da estrutura HTML
- Inserção de elementos da interface
- Uso de IA para geração de imagens

---

### Aula 2 - Estruturação do projeto

- Estruturação mais profissional do HTML
- Boas práticas de organização de código
- Uso da IA como apoio no desenvolvimento

---

### Aula 3 - Interatividade e refinamento

- Introdução ao JavaScript
- Implementação de Light/Dark Mode
- Criação de variáveis CSS
- Responsividade com apoio de IA
- Melhoria da organização do código

---

### Aula 4 - Finalização

- Análise de qualidade com Lighthouse
- Criação do repositório
- Deploy do projeto
- Publicação na web
- Transformação em projeto de portfólio

---

## 🧠 Aprendizados

- Uso da Inteligência Artificial como ferramenta de produtividade
- Como estruturar melhor um projeto Front-End
- Importância da organização e semântica do código
- Boas práticas de desenvolvimento e melhoria contínua
